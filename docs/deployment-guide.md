# 云服务器部署准备指南（小白版）

> 本文档面向没有服务器运维经验的开发者，手把手引导完成从购买服务器到部署上线的全流程。

---

## 第一步：购买域名（1天）

### 为什么需要域名？
- 备案必须用域名（不能只通过IP访问）
- 给用户一个友好的访问地址，如 `https://jiankang.example.com`

### 操作步骤
1. 打开 [阿里云域名注册](https://wanwang.aliyun.com/) 或 [腾讯云域名注册](https://dnspod.cloud.tencent.com/)
2. 搜索你想要的域名（如 `xxxjiankang.com`），选择一个未被注册的
3. 加入购物车，选择 **1年** 期限
4. 完成支付（.com 域名约 60-80元/年，.cn 约 30元/年）
5. 进入 **域名控制台** → 点击 **实名认证**
   - 个人用户：上传身份证正反面照片
   - 审核时间：通常 **1小时内** 通过

> ⚠️ **重要**：域名必须实名认证后才能进行下一步的 ICP 备案！

---

## 第二步：购买云服务器（10分钟）

### 推荐配置（内测阶段）
```
CPU: 2核
内存: 4GB
硬盘: 80GB SSD
带宽: 3Mbps（约 375KB/s，够用了）
操作系统: Ubuntu 22.04 LTS（64位）
地域: 选择离你最近的城市（如华东1-杭州、华南1-深圳）
```

### 操作步骤（以阿里云为例）
1. 打开 [阿里云ECS购买页](https://ecs-buy.aliyun.com/)
2. 选择 **按量付费**（前期测试用，随时可转包年包月）或 **包年包月**
3. 按以下配置选择：
   - **地域**：选离你最近的
   - **实例规格**：共享标准型 s6 / 2核4G
   - **镜像**：公共镜像 → Ubuntu → 22.04 64位
   - **存储**：系统盘 80GB SSD
   - **带宽**：按固定带宽 3Mbps
   - **安全组**：先选默认的，后面再改
4. 点击 **立即购买** → 支付
5. 支付完成后，进入 **ECS控制台** → 找到你的实例

### 记录以下信息（后面都要用到）
- [ ] 服务器公网IP（如 `123.45.67.89`）
- [ ]  root 密码（购买时设置的，忘了可以在控制台重置）
- [ ] 服务器所在地域

## 第三步：ICP备案（7-20天，最关键！）

### 什么是ICP备案？
在中国大陆，任何网站要想通过域名访问，必须向工信部登记。这是法律要求，没有备案的域名会被云服务商拦截。

### 备案期间能做什么？
- ✅ 可以通过 **服务器IP地址** 直接访问（临时测试）
- ✅ 继续开发、测试功能
- ❌ 不能通过域名访问（会被阻断）

### 操作步骤（以阿里云为例）
1. 登录 [阿里云ICP备案系统](https://beian.aliyun.com/)
2. 点击 **开始备案** → 填写信息：
   - **备案主体**：个人（选"个人"）
   - **证件类型**：身份证
   - **域名**：填写你刚买的域名
   - **网站名称**：不能出现"医疗""医院""诊断"等敏感词，建议用"个人健康数据记录"
   - **网站备注**："本网站为个人健康数据管理工具，仅用于记录个人健康指标，不提供医疗诊断服务"
3. 上传资料：
   - 身份证正反面
   - 手持身份证照片（按模板拍）
   - 《网站备案信息真实性核验单》（在线签署）
4. 阿里云初审（1-2个工作日）
5. 工信部短信核验（你会收到短信，24小时内完成验证）
6. 管局审核（7-15个工作日）
7. 备案成功！你会收到邮件和短信通知

> 💡 **建议**：买完服务器和域名后**立即提交备案**，备案期间继续开发和测试，不要等！

## 第四步：连接你的服务器（5分钟）

### 在 Mac 上连接
打开终端（Terminal），输入：
```bash
ssh root@你的服务器IP
# 例如：ssh root@123.45.67.89
```
然后输入你设置的 root 密码。

### 在 Windows 上连接
1. 下载 [PuTTY](https://www.putty.org/) 或 Windows Terminal
2. 打开后输入服务器IP，端口 22，点击连接
3. 输入用户名 `root` 和密码

### 第一次登录后必做
```bash
# 1. 更新系统软件包
apt update && apt upgrade -y

# 2. 安装必要工具
apt install -y curl wget git vim ufw

# 3. 修改 root 密码（如果太简单）
passwd

# 4. 设置时区为上海
timedatectl set-timezone Asia/Shanghai

# 5. 查看当前时间是否正确
date
```

## 第五步：安装 Docker + Docker Compose（10分钟）

服务器连接成功后，一键安装：

```bash
# 安装 Docker
curl -fsSL https://get.docker.com | bash

# 启动 Docker 并设为开机自启
systemctl start docker
systemctl enable docker

# 安装 Docker Compose
apt install -y docker-compose

# 验证安装
docker --version
docker-compose --version
```

## 第六步：配置安全组（防火墙）

### 为什么要做？
默认所有端口都开放，容易被黑客扫描攻击。只开放必要的端口。

### 操作步骤
1. 进入阿里云/腾讯云控制台 → 找到你的ECS实例
2. 点击 **安全组** → **配置规则** → **入方向**
3. 添加以下规则：

| 类型 | 端口 | 授权对象 | 说明 |
|------|------|---------|------|
| SSH | 22 | 你的家庭IP/32 | 只允许你自己连接服务器 |
| HTTP | 80 | 0.0.0.0/0 | 所有人可访问网站 |
| HTTPS | 443 | 0.0.0.0/0 | 所有人可访问安全网站 |

> 💡 **如何查看你的家庭IP？** 在百度搜 "IP地址"，显示的第一个就是你当前网络的公网IP。

## 准备完成后的检查清单

在购买服务器并执行完上述步骤后，请确认：

- [ ] 域名已购买并完成实名认证
- [ ] 云服务器已购买并记录公网IP
- [ ] ICP备案已提交（进入审核中状态）
- [ ] 能通过 SSH 连接到服务器
- [ ] Docker 和 Docker Compose 已安装
- [ ] 安全组已配置（22、80、443端口）
- [ ] 服务器系统时间正确（`date` 命令显示北京时间）
