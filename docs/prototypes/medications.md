# 页面原型 - 用药管理

**页面名称**: Medications / 用药管理
**页面路径**: `/medications`, `/medications/new`, `/medications/logs`
**版本**: v1.0.0

---

## 1. 页面概述

用药管理是 PRD v1.1.0 新增的核心功能模块，包含用药提醒设置和用药记录两大功能。使用紫色 (#722ED1) 作为主色调，与医疗蓝区分。

### 1.2 包含页面

1. **用药管理首页** (`/medications`) - 用药设置列表 + 今日记录
2. **添加用药** (`/medications/new`) - 设置新的用药提醒
3. **用药记录** (`/medications/logs`) - 历史服药记录时间线
4. **编辑用药** (`/medications/:id/edit`) - 修改用药设置

### 3.2 顶部导航栏

**左侧**: 返回按钮
**中间**: 页面标题 "添加用药" / "编辑用药"
**右侧**: "保存" 按钮 (紫色主按钮样式)

### 3.3 药品信息

**药品名称**:
- 标签: "药品名称 *"（必填）
- 输入框: 支持搜索和选择
- 常用药品选项:
  - 环孢素
  - 他克莫司
  - 雷帕霉素
  - 钙片
  - 铁剂
  - 骨化三醇
  - 降压药（多种）
  - 其他（自定义输入）

**规格**:
- 标签: "规格"
- 输入框: 占位符 "如: 50mg/片"
- 常用规格快捷选择

**每次剂量**:
- 标签: "每次剂量 *"（必填）
- 数值输入 + 单位选择
- 单位选项: 片、粒、mg、ml、袋

### 3.4 服药设置

**服药频率**:
- 标签: "服药频率 *"（必填）
- 选择器选项:
  - 每日1次
  - 每日2次
  - 每日3次
  - 每日4次
  - 隔日1次
  - 每周1次
  - 每周2次
  - 按需服用（不提醒）

**服药时间**:
- 标签: "服药时间 *"（必填）
- 时间选择器列表
- 支持添加多个时间（根据频率）
- 每个时间可删除

**提前提醒**:
- 标签: "提前提醒"
- 选择器选项:
  - 准时
  - 提前5分钟
  - 提前10分钟
  - 提前15分钟
  - 提前30分钟

### 3.5 提醒设置

**开启用药提醒**:
- 开关控制
- 默认开启

**重复提醒**:
- 复选框: "错过30分钟后再次提醒"
- 默认勾选
- 说明: "最多提醒3次"

**提醒方式**:
- 单选框组
- 选项:
  - 应用内通知（默认）
  - 短信提醒（标记为VIP功能）
  - 微信推送（标记为VIP功能）

### 3.6 表单校验

**必填项**:
- 药品名称
- 每次剂量
- 服药频率
- 服药时间

**校验规则**:
- 药品名称: 2-50字符
- 剂量: 大于0的数字
- 时间: 不能重复

## 8. 交互说明

### 8.1 开关用药提醒

1. 用户点击开关
2. 显示确认弹窗（关闭时）
3. 更新状态
4. Toast 提示 "已开启提醒" / "已关闭提醒"

### 8.2 删除用药设置

1. 左滑显示删除按钮
2. 点击删除
3. 确认弹窗: "确定删除这个用药提醒？删除后将不再提醒"
4. 确认后删除

### 8.3 保存用药设置

1. 校验表单
2. 显示加载状态
3. 保存成功:
   - Toast 提示 "保存成功"
   - 返回列表页
   - 如开启提醒，显示"将在XX时间提醒您"
4. 保存失败: 显示错误信息
