# 页面原型 - 预警中心

**页面名称**: Alerts / 预警中心
**页面路径**: `/alerts`, `/alerts/:id`
**版本**: v1.0.0

---

## 1. 页面概述

预警中心集中展示所有健康预警信息，包括指标异常、漏服提醒等。按照严重程度分级显示，帮助用户及时发现和处理健康问题。

### 1.1 预警级别

| 级别 | 颜色 | 触发条件 | 处理方式 |
|------|------|----------|----------|
| **Critical** | 红色 `#F5222D` | 危及生命 | 弹窗+短信提醒 |
| **Warning** | 橙色 `#FAAD14` | 明显异常 | 首页卡片提醒 |
| **Info** | 蓝色 `#1890FF` | 轻微波动 | 列表展示 |

### 1.2 包含页面

1. **预警列表页** (`/alerts`) - 所有预警列表
2. **预警详情页** (`/alerts/:id`) - 单条预警详情

## 4. 预警规则说明

### 4.1 指标异常预警

**严重高钾血症 (Critical)**:
- 条件: 血钾 > 6.0 mmol/L
- 通知: 弹窗 + 声音 + 短信
- 建议: 立即就医

**高钾血症 (Warning)**:
- 条件: 血钾 > 5.5 mmol/L
- 通知: 应用内通知
- 建议: 控制饮食

**肌酐突增 (Warning)**:
- 条件: 较基线上升 > 20%
- 通知: 应用内通知
- 建议: 复查就医

**高尿酸血症 (Warning)**:
- 条件: 尿酸 > 420 μmol/L
- 通知: 应用内通知
- 建议: 控制饮食

**血药浓度异常 (Warning)**:
- 条件: 超出目标范围
- 通知: 应用内通知
- 建议: 咨询医生

### 4.2 漏服提醒

**漏服提醒 (Warning)**:
- 条件: 超过计划时间 30 分钟未记录
- 通知: 应用内通知 + 首页卡片
- 重复: 每 30 分钟一次，最多 3 次
- 建议: 尽快补服或咨询医生

### 4.3 信息提示

**贫血提示 (Info)**:
- 条件: 血红蛋白 < 110 g/L
- 通知: 列表展示
- 建议: 复查血常规

## 6. 交互说明

### 6.1 标记已读

**方式一**: 点击详情页底部按钮
**方式二**: 列表页左滑"标记已读"
**方式三**: 点击"全部已读"

**效果**:
- 移除"新"标记
- 更新未读数量角标
- 卡片背景变浅色

### 6.2 删除预警

1. 列表页左滑显示删除按钮
2. 点击删除
3. 确认弹窗: "确定删除这条预警记录？"
4. 确认后删除

### 6.3 预警推送

**应用内推送**:
- 从顶部滑入通知卡片
- 点击跳转到详情页

**后台推送**:
- 系统通知
- 点击打开应用并跳转到详情页
