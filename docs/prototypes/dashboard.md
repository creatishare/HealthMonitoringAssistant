# 页面原型 - 首页仪表盘

**页面名称**: Dashboard / 首页仪表盘
**页面路径**: `/dashboard`
**版本**: v1.0.0

---

## 1. 页面概述

首页仪表盘是用户打开应用后看到的第一个页面，集中展示今日待办、用药提醒、最近指标和预警信息，是用户每日使用频率最高的页面。

### 1.1 设计目标

- 一眼看到今日最重要的健康信息
- 快速完成每日打卡任务
- 及时处理用药提醒和预警
- 简洁清晰，减少认知负担

### 1.2 用户场景

1. **晨起场景**: 查看今日用药、记录体重血压
2. **餐后场景**: 确认已服药
3. **睡前场景**: 查看今日数据是否记录完整
4. **收到提醒后**: 快速进入确认服药

## 3. 模块详细设计

### 3.1 顶部欢迎栏

**布局**: 左右布局
**高度**: 56px
**背景**: 白色
**内边距**: 16px

**左侧**:
- 问候语: "上午好，张先生"
- 字号: 20px, 600
- 颜色: `#262626`

**右侧**:
- 日期: "2026年3月29日 周六"
- 字号: 14px
- 颜色: `#595959`

**交互**:
- 点击头像区域进入个人中心

---

### 3.2 今日打卡区

**布局**: 横向滚动/网格
**背景**: 白色卡片
**内边距**: 16px
**圆角**: 12px
**阴影**: `0 2px 8px rgba(0,0,0,0.08)`

**标题**: "今日打卡"
- 字号: 16px, 500
- 颜色: `#262626`
- 底部间距: 12px

**打卡项设计**:

每个打卡项:
- 宽度: 80px
- 高度: 80px
- 圆角: 12px
- 背景: `#F5F5F5`（未完成）/ `#E6F7FF`（已完成）
- 图标: 32px
- 标签: 14px, 居中

**打卡项列表**:

| 图标 | 名称 | 状态 | 点击行为 |
|------|------|------|----------|
| ⚖️ | 体重 | 未录入/已录入 | 弹出体重录入弹窗 |
| 🩺 | 血压 | 未录入/已录入 | 弹出血压录入弹窗 |
| 💧 | 饮水 | 未达标/已达标 | 弹出饮水记录弹窗 |
| ➕ | 更多 | - | 跳转到指标录入页 |

**已完成状态**:
- 背景: `#E6F7FF`
- 图标颜色: `#1890FF`
- 显示勾选标记

**交互**:
- 点击未完成项: 弹出快速录入弹窗
- 点击已完成项: 显示今日已录数据
- 左滑查看更多打卡项

---

### 3.3 用药提醒区

**布局**: 垂直列表
**背景**: 白色卡片
**内边距**: 16px
**圆角**: 12px
**阴影**: `0 2px 8px rgba(0,0,0,0.08)`

**标题栏**:
- 左侧: ⏰ 图标 + "今日用药"
- 右侧: 进度 "3/5" + 展开箭头
- 字号: 16px, 500
- 颜色: `#262626`

**用药卡片设计**:

每个用药项:
- 高度: 64px
- 左边框: 4px solid `#722ED1`
- 背景: `linear-gradient(to right, #F9F0FF, #FFFFFF)`
- 内边距: 12px 16px
- 圆角: 8px
- 底部间距: 8px

**用药项内容**:
```
┌─────────────────────────────────────────┐
│ ● 08:00    环孢素    2片    [已服 ✓]   │
│            50mg/片                      │
└─────────────────────────────────────────┘
```

**状态样式**:

| 状态 | 左边框 | 背景 | 状态标记 |
|------|--------|------|----------|
| 已服药 | `#52C41A` | `#F6FFED` | 绿色勾选 |
| 待服药 | `#722ED1` | `#F9F0FF` | 紫色时钟 |
| 已跳过 | `#BFBFBF` | `#F5F5F5` | 灰色划线 |

**交互**:
- 点击待服药项: 弹出确认服药弹窗
  - 显示药物信息
  - "确认已服药" 按钮（紫色 `#722ED1`）
  - "跳过本次" 按钮
  - 记录实际服药时间（默认当前时间，可修改）
- 点击已服药项: 显示服药详情
- 点击"查看全部": 跳转到用药管理页

---

### 3.4 预警提醒区

**布局**: 垂直列表
**显示条件**: 有未读预警时显示

**预警卡片设计**:

根据级别不同样式:

**Critical (危险)**:
- 背景: `#FFF2F0`
- 左边框: 4px solid `#F5222D`
- 图标: 红色警告三角形
- 标题: 16px, `#F5222D`
- 内容: 14px, `#262626`
- 建议: 14px, `#595959`

**Warning (警告)**:
- 背景: `#FFFBE6`
- 左边框: 4px solid `#FAAD14`
- 图标: 橙色警告
- 标题: 16px, `#FAAD14`

**Info (信息)**:
- 背景: `#E6F7FF`
- 左边框: 4px solid `#1890FF`
- 图标: 蓝色信息
- 标题: 16px, `#1890FF`

**示例卡片**:
```
┌─────────────────────────────────────────┐
│ ⚠️ 肌酐突增预警                         │
│                                         │
│ 肌酐较上次上升25%，建议及时就医复查      │
│                                         │
│ 💡 建议：请检查是否按时服药，控制蛋白质  │
│    摄入                                  │
│                                         │
│ [查看详情]           [我知道了]          │
└─────────────────────────────────────────┘
```

**交互**:
- 点击卡片: 跳转到预警详情页
- 点击"我知道了": 标记为已读，收起卡片
- 左滑: 删除/标记已读

### 3.5 最近指标区

**布局**: 2列网格
**背景**: 透明
**标题**: "最近指标" + 右侧"查看全部"链接

**指标卡片设计**:

每个卡片:
- 宽度: 计算 (50% - 8px)
- 高度: 120px
- 背景: 白色
- 圆角: 12px
- 内边距: 16px
- 阴影: `0 2px 8px rgba(0,0,0,0.08)`

**卡片内容**:
```
┌──────────────────┐
│ 肌酐             │  ← 指标名称 14px, #595959
│                  │
│ 180              │  ← 数值 28px, 600, 颜色根据状态
│ ⚠️ 偏高          │  ← 状态标签 12px
│                  │
│ 3月25日          │  ← 日期 12px, #8C8C8C
└──────────────────┘
```

**状态颜色**:
- 正常: `#52C41A`
- 警告: `#FAAD14`
- 危险: `#F5222D`

**显示指标** (最多4个):
1. 肌酐
2. 尿素氮
3. 血钾
4. 尿酸 (新增)

**交互**:
- 点击卡片: 跳转到该指标趋势图
- 点击"查看全部": 跳转到指标列表页

## 6. 响应式适配

### 6.1 小屏手机 (<375px)

- 指标卡片改为单列显示
- 打卡项宽度调整为 72px
- 减少卡片内边距至 12px

### 6.2 大屏手机 (>414px)

- 指标卡片可显示 3 列
- 增加模块间距至 20px
- 用药卡片显示更多信息

## 7. 状态汇总

| 状态 | 视觉表现 |
|------|----------|
| 页面加载中 | 骨架屏占位 |
| 数据加载中 | 卡片内旋转动画 |
| 网络错误 | 错误提示 + 重试按钮 |
| 无数据 | 空状态插图 |
| 有预警 | 预警卡片置顶 |
