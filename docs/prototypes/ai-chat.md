# 页面原型 - AI 助手对话

**页面名称**: AI Chat / AI 助手
**页面路径**: `/ai-chat`
**版本**: v1.0.0

---

## 1. 页面概述

AI 助手对话页面提供智能问答服务，帮助用户理解健康指标、获取饮食建议、了解疾病知识等。AI 助手基于用户的健康数据提供个性化建议。

### 1.1 功能范围

**支持的功能**:
- 指标解读（基于用户实际数据）
- 饮食建议
- 用药指导（一般性）
- 疾病知识问答
- 化验单解读

**明确拒绝**:
- 具体诊疗建议
- 药物剂量调整
- 诊断疾病

## 7. 功能示例对话

### 7.1 指标解读示例

**用户**: "我的肌酐最近有点高"

**AI**:
```
我查看了您最近的记录：

[数据卡片]
肌酐: 180 μmol/L (偏高)
参考范围: 44-133
较基线: ↑ 25 (16%)

分析：
肌酐较您的基线值（155）有所上升。

可能原因：
1. 蛋白质摄入过多
2. 水分控制不佳
3. 近期感染或劳累
4. 药物影响

建议：
• 回顾近期饮食，控制蛋白质摄入
• 确保按时服药
• 建议及时就医复查

⚠️ 免责声明：以上建议仅供参考，
请遵医嘱，如有不适请及时就医。
```

### 7.2 饮食建议示例

**用户**: "血钾高吃什么好？"

**AI**:
```
针对血钾偏高的情况，建议：

✅ 推荐食物（低钾）：
• 水果：苹果、梨、蓝莓、草莓
• 蔬菜：白菜、黄瓜、洋葱、豆芽
• 主食：米饭、面条、馒头
• 蛋白质：鸡蛋、鸡胸肉（适量）

❌ 限制食物（高钾）：
• 水果：香蕉、橙子、猕猴桃、榴莲
• 蔬菜：菠菜、土豆、蘑菇、番茄
• 其他：坚果、巧克力、运动饮料

💡 降钾技巧：
• 蔬菜切小块，浸泡换水后再烹饪
• 水煮后弃汤食用
• 避免饮用菜汤、果汁

⚠️ 免责声明：以上建议仅供参考，
具体饮食方案请咨询营养师或医生。
```

### 7.3 用药提醒示例

**用户**: "环孢素什么时候吃？"

**AI**:
```
关于环孢素的服用：

⏰ 服用时间：
• 通常每日2次，间隔12小时
• 建议固定时间（如早8点、晚8点）
• 空腹或餐后2小时服用

📋 注意事项：
• 整片吞服，不要咀嚼
• 葡萄柚汁会影响药效，避免同服
• 定期监测血药浓度

🩸 血药浓度监测：
• C0（谷浓度）：服药前抽血
• 目标范围：100-200 ng/mL
• 根据浓度调整剂量（由医生决定）

⚠️ 重要提示：
请勿自行调整剂量或停药，
所有调整需在医生指导下进行。
```

## 8. 边界处理

### 8.1 超出范围的问题

**用户**: "我应该吃什么药？"

**AI**:
```
抱歉，我无法为您推荐具体药物。

原因：
• 药物选择需要专业医生根据您的
  具体情况判断
• 错误的用药可能带来风险

建议：
• 请前往肾内科就诊
• 携带您的检查报告
• 由医生制定治疗方案

我可以帮您：
• 了解药物的一般知识
• 提醒用药时间
• 记录用药情况
```

### 8.2 紧急情况

**用户**: "我胸闷气短怎么办？"

**AI**:
```
🚨 这属于紧急症状，请立即就医！

请立即：
• 拨打急救电话 120
• 或前往最近医院急诊

不要等待我的回复，
您的健康比任何建议都重要。

⚠️ 胸闷气短可能涉及心脏问题，
需要专业医生立即评估。
```

## 10. 交互说明

### 10.1 发送消息

1. 用户输入问题
2. 点击发送或按回车
3. 用户消息显示在对话区
4. 显示 AI 思考中状态
5. AI 回复显示
6. 自动滚动到底部

### 10.2 历史记录

- 保存最近 30 天对话
- 支持查看历史
- 支持清空当前对话

### 10.3 错误处理

**网络错误**:
- 显示: "网络连接失败，请检查后重试"
- 提供重试按钮

**服务繁忙**:
- 显示: "服务繁忙，请稍后再试"
- 提供重试按钮
