## 1. 指标定义

### 1.1 基础生化指标

#### 血清肌酐 (Creatinine)
```yaml
字段名: creatinine
显示名称: 血清肌酐
单位: μmol/L
单位缩写: μmol/L
正常范围: [44, 133]
CKD3期阈值: ">133"
CKD4期阈值: ">300"
危险阈值: ">500"
采集方式: 静脉血
采集时间: 透析前后均可，需保持一致
临床意义: 反映肾功能的主要指标
注意事项: 肌肉量影响基线值，需建立个人基线
```

#### 尿素氮 (Urea/BUN)
```yaml
字段名: urea
显示名称: 尿素氮
单位: mmol/L
单位缩写: mmol/L
正常范围: [2.6, 7.5]
CKD患者常见范围: [10, 25]
危险阈值: ">30"
采集方式: 静脉血
临床意义: 反映蛋白质代谢和肾功能
饮食关联: 高蛋白饮食会升高
```

#### 估算肾小球滤过率 (eGFR)
```yaml
字段名: egfr
显示名称: eGFR
单位: mL/min/1.73m²
单位缩写: mL/min
计算公式: CKD-EPI公式（基于肌酐）
正常范围: [90, 120]
CKD分期依据:
  G1: ">=90" (正常或增高)
  G2: "60-89" (轻度下降)
  G3a: "45-59" (轻到中度下降)
  G3b: "30-44" (中到重度下降)
  G4: "15-29" (重度下降)
  G5: "<15" (肾衰竭)
采集方式: 计算值
临床意义: 评估肾功能分期的金标准
```

### 1.2 电解质指标

#### 血钾 (Potassium)
```yaml
字段名: potassium
显示名称: 血钾
单位: mmol/L
单位缩写: mmol/L
正常范围: [3.5, 5.3]
警告阈值: ">5.5"
危险阈值: ">6.0"
紧急阈值: ">6.5"
采集方式: 静脉血
临床意义: 电解质平衡，影响心脏功能
饮食控制: 高钾食物限制（香蕉、橙子、土豆等）
紧急处理: ">6.0需立即就医"
```

#### 血钠 (Sodium)
```yaml
字段名: sodium
显示名称: 血钠
单位: mmol/L
单位缩写: mmol/L
正常范围: [136, 145]
低钠阈值: "<135"
高钠阈值: ">145"
采集方式: 静脉血
临床意义: 水平衡和渗透压调节
```

#### 血磷 (Phosphorus)
```yaml
字段名: phosphorus
显示名称: 血磷
单位: mmol/L
单位缩写: mmol/L
正常范围: [0.87, 1.45]
CKD患者目标: "<1.78"
高磷阈值: ">1.78"
采集方式: 静脉血
临床意义: 骨代谢和血管钙化风险
饮食控制: 限制高磷食物，配合磷结合剂
```

### 1.3 血液指标

#### 血红蛋白 (Hemoglobin)
```yaml
字段名: hemoglobin
显示名称: 血红蛋白
单位: g/L
单位缩写: g/L
正常范围（男）: [120, 160]
正常范围（女）: [110, 150]
贫血阈值: "<110"
严重贫血: "<60"
采集方式: 静脉血
临床意义: 肾性贫血监测
CKD患者目标: "110-120"
治疗: 促红细胞生成素(EPO)、铁剂
```

### 1.4 新增指标 (PRD v1.1.0)

#### 尿酸 (Uric Acid)
```yaml
字段名: uric_acid
显示名称: 尿酸
单位: μmol/L
单位缩写: μmol/L
正常范围（男）: [150, 416]
正常范围（女）: [89, 357]
警告阈值: ">420"
高尿酸血症: ">420"
痛风风险: ">480"
采集方式: 静脉血
临床意义: 痛风风险评估
饮食建议: 低嘌呤饮食，多饮水
关联疾病: 痛风、肾结石、心血管风险
```

#### 血药浓度 (Drug Concentration)
```yaml
字段名: drug_concentration
显示名称: 血药浓度
单位: ng/mL
单位缩写: ng/mL
药物类型:
  cyclosporine_c0:
    名称: 环孢素C0（谷浓度）
    采集时间: 服药前（下次服药前）
    目标范围: [100, 200]
    低值风险: 排斥反应
    高值风险: 肾毒性

  cyclosporine_c2:
    名称: 环孢素C2（峰浓度）
    采集时间: 服药后2小时
    目标范围: [700, 1200]
    备注: 部分医院采用

  tacrolimus_c0:
    名称: 他克莫司C0（谷浓度）
    采集时间: 服药前
    目标范围: [5, 15]
    低值风险: 排斥反应
    高值风险: 肾毒性、神经毒性

  sirolimus_c0:
    名称: 雷帕霉素C0（谷浓度）
    采集时间: 服药前
    目标范围: [5, 15]
    备注: 移植后维持期

采集方式: 静脉血
关联信息:
  - 药物类型
  - 上次服药时间
  - 采血时间
  - 与服药间隔时长
临床意义: 指导免疫抑制剂剂量调整
注意事项:
  - 采血时间必须准确记录
  - 与服药记录关联分析
  - 需医生解读调整方案
```

### 1.5 日常监测指标

#### 体重 (Weight)
```yaml
字段名: weight
显示名称: 体重
单位: kg
单位缩写: kg
透析患者:
  干体重: 透析后理想体重
  超滤量: 本次透析脱水量
  体重增长: 透析间期体重增长
警告阈值: ">干体重*1.05" (增长超5%)
危险阈值: ">干体重*1.07" (增长超7%)
采集方式: 体重秤
测量时间: 固定时间（如晨起空腹）
临床意义: 水分控制评估
```

#### 血压 (Blood Pressure)
```yaml
字段名:
  systolic: 收缩压
  diastolic: 舒张压
显示名称: 血压
单位: mmHg
单位缩写: mmHg
正常范围:
  收缩压: [90, 140]
  舒张压: [60, 90]
高血压:
  收缩压: ">=140"
  舒张压: ">=90"
危险高血压:
  收缩压: ">=180"
  舒张压: ">=110"
低血压:
  收缩压: "<90"
采集方式: 电子血压计
测量时间: 固定时间，建议早晚各一次
临床意义: 心血管风险评估
```

#### 血糖 (Blood Glucose)
```yaml
字段名: blood_glucose
显示名称: 血糖
单位: mmol/L
单位缩写: mmol/L
空腹正常: [3.9, 6.1]
空腹受损: [6.1, 7.0]
糖尿病: ">=7.0"
餐后2h正常: "<7.8"
餐后2h糖尿病: ">=11.1"
低血糖: "<3.9"
采集方式: 血糖仪
测量时间: 空腹、餐后2h、随机
临床意义: 糖尿病管理
```

#### 尿量 (Urine Output)
```yaml
字段名: urine_output
显示名称: 尿量
单位: mL/24h
单位缩写: mL
正常范围: [1000, 2000]
少尿: "<400"
无尿: "<100"
采集方式: 量杯测量
测量时间: 24小时总量
临床意义: 残余肾功能评估
```

## 2. 预警规则

### 2.1 严重预警 (Critical)

```yaml
rules:
  - id: hyperkalemia_critical
    名称: 严重高钾血症
    条件: potassium > 6.0
    级别: critical
    颜色: "#F5222D"
    消息: "血钾严重超标（{value} mmol/L），请立即联系医生或前往急诊！"
    建议: "高钾可能导致心律失常，请勿拖延"
    通知方式: 弹窗 + 短信 + 声音提醒

  - id: severe_hypertension
    名称: 严重高血压
    条件: systolic >= 180 OR diastolic >= 110
    级别: critical
    颜色: "#F5222D"
    消息: "血压严重升高（{value} mmHg），请立即就医！"
    建议: "避免剧烈运动，保持安静，及时就医"
    通知方式: 弹窗 + 短信

  - id: severe_hypoglycemia
    名称: 严重低血糖
    条件: blood_glucose < 3.0
    级别: critical
    颜色: "#F5222D"
    消息: "血糖严重偏低（{value} mmol/L），请立即补充糖分！"
    建议: "立即进食糖果、糖水等，如意识不清请拨打急救电话"
    通知方式: 弹窗 + 声音提醒
```

### 2.2 警告预警 (Warning)

```yaml
rules:
  - id: hyperkalemia_warning
    名称: 高钾血症
    条件: potassium > 5.5
    级别: warning
    颜色: "#FAAD14"
    消息: "血钾偏高（{value} mmol/L），请注意饮食控制"
    建议: "减少高钾食物摄入，如香蕉、橙子、土豆等"
    通知方式: 首页卡片提醒

  - id: hyperuricemia_warning
    名称: 高尿酸血症
    条件: uric_acid > 420
    级别: warning
    颜色: "#FAAD14"
    消息: "尿酸偏高（{value} μmol/L），注意控制饮食，多饮水"
    建议: "减少高嘌呤食物摄入，如动物内脏、海鲜、浓肉汤"
    通知方式: 首页卡片提醒

  - id: drug_concentration_low
    名称: 血药浓度偏低
    条件: drug_concentration < 目标范围下限
    级别: warning
    颜色: "#FAAD14"
    消息: "{drug_name}血药浓度偏低（{value} ng/mL），可能影响疗效"
    建议: "请检查是否按时服药，建议咨询医生"
    通知方式: 首页卡片提醒

  - id: drug_concentration_high
    名称: 血药浓度偏高
    条件: drug_concentration > 目标范围上限
    级别: warning
    颜色: "#FAAD14"
    消息: "{drug_name}血药浓度偏高（{value} ng/mL），请注意观察"
    建议: "请勿自行调整药物剂量，建议咨询医生"
    通知方式: 首页卡片提醒

  - id: creatinine_rise
    名称: 肌酐突增
    条件: creatinine > 基线值 * 1.2
    级别: warning
    颜色: "#FAAD14"
    消息: "肌酐较基线上升{percent}%，建议及时就医复查"
    建议: "请检查是否按时服药，控制蛋白质摄入"
    通知方式: 首页卡片提醒

  - id: weight_gain_warning
    名称: 体重增长过快
    条件: dialysis == true AND weight_gain > dry_weight * 0.05
    级别: warning
    颜色: "#FAAD14"
    消息: "透析间期体重增长超过5%，请控制水分摄入"
    建议: "控制每日饮水量，避免高盐食物"
    通知方式: 首页卡片提醒

  - id: hypertension_warning
    名称: 血压升高
    条件: systolic >= 140 OR diastolic >= 90
    级别: warning
    颜色: "#FAAD14"
    消息: "血压偏高（{value} mmHg），请注意监测"
    建议: "按时服用降压药，减少盐分摄入"
    通知方式: 首页卡片提醒

  - id: missed_medication_warning
    名称: 漏服提醒
    条件: scheduled_time_passed > 30min AND not_recorded
    级别: warning
    颜色: "#722ED1"
    消息: "您已错过{drug_name}的服药时间，请尽快补服或咨询医生"
    建议: "规律服药对控制病情非常重要"
    通知方式: 应用内通知 + 首页卡片
    重复提醒: 每30分钟一次，最多3次
```

## 3. 饮食指导

### 3.1 水分控制

```yaml
透析患者:
  透析间期体重增长: "<干体重的3-5%"
  每日水分摄入: 尿量 + 500mL
  测量频率: 每日同一时间
  记录内容: 体重、尿量、饮水量

非透析患者:
  根据尿量调整: 前日尿量 + 500mL
  水肿者: 严格限制
```

### 3.2 钠摄入控制

```yaml
每日限制: "<2000mg" (约5g盐)
高钠食物避免:
  - 腌制食品（咸菜、腊肉）
  - 加工食品（火腿、香肠）
  - 方便食品（方便面、罐头）
  - 高盐调味品（酱油、味精）
低钠替代品: 醋、柠檬汁、香料
```

### 3.3 钾摄入控制

```yaml
每日限制: "<2000mg" (血钾高者)
高钾食物限制:
  水果: 香蕉、橙子、猕猴桃、榴莲
  蔬菜: 菠菜、土豆、蘑菇、番茄
  其他: 坚果、巧克力、运动饮料
降钾技巧:
  - 蔬菜切小块浸泡换水
  - 水煮后弃汤食用
  - 避免菜汤、果汁
```

### 3.4 磷摄入控制

```yaml
每日限制: "<1000mg"
高磷食物限制:
  - 奶制品（牛奶、奶酪）
  - 动物内脏
  - 坚果类
  - 碳酸饮料
  - 加工食品（含磷添加剂）
磷结合剂: 餐中服用，随餐咀嚼
```

### 3.5 蛋白质摄入

```yaml
透析患者:
  每日摄入: "1.0-1.2g/kg体重"
  优质蛋白比例: ">50%"
  推荐: 鸡蛋、瘦肉、鱼、牛奶

非透析CKD患者:
  每日摄入: "0.6-0.8g/kg体重"
  优质蛋白为主
```

### 3.6 尿酸控制饮食

```yaml
高嘌呤食物避免:
  - 动物内脏（肝、肾、脑）
  - 海鲜（沙丁鱼、凤尾鱼、贝类）
  - 浓肉汤、肉汁
  - 酒精（尤其是啤酒）

中嘌呤食物适量:
  - 肉类（每日不超过100g）
  - 部分鱼类
  - 豆制品

推荐食物:
  - 低脂奶制品
  - 蔬菜（除高嘌呤蔬菜）
  - 水果
  - 多饮水（每日2000mL以上，无禁忌者）
```

## 4. 用药规范

### 4.1 常用药物分类

```yaml
降压药物:
  - ACEI/ARB类（保护肾功能）
  - 钙通道阻滞剂
  - β受体阻滞剂

纠正贫血:
  - 促红细胞生成素(EPO)
  - 铁剂

调节钙磷:
  - 磷结合剂（餐中服用）
  - 活性维生素D

免疫抑制剂（移植患者）:
  - 环孢素
  - 他克莫司
  - 雷帕霉素
  - 吗替麦考酚酯
```

### 4.2 服药时间规范

```yaml
空腹服用: 餐前1小时或餐后2小时
餐中服用: 与食物同服（如磷结合剂）
餐后服用: 减少胃肠道刺激
睡前服用: 某些降压药
定时服用: 免疫抑制剂需严格定时
```

### 4.3 血药浓度监测要点

```yaml
监测频率:
  移植初期: 每周1-2次
  稳定期: 每月1次
  调整剂量后: 3-5天后复查

采血时间要求:
  C0（谷浓度）: 下次服药前即刻
  C2（峰浓度）: 服药后2小时（±15分钟）

记录要点:
  - 准确服药时间
  - 采血时间
  - 与服药间隔
  - 药物剂型/剂量变化
```

## 5. 数据采集规范

### 5.1 录入时间建议

| 指标 | 建议采集时间 | 频率 |
|------|-------------|------|
| 体重 | 晨起空腹、睡前 | 每日 |
| 血压 | 晨起后、睡前 | 每日2次 |
| 血糖 | 空腹、餐后2h | 遵医嘱 |
| 尿量 | 24小时总量 | 每日 |
| 血常规 | - | 每月1-2次 |
| 生化全套 | - | 每月1次 |
| 血药浓度 | 遵医嘱 | 初期每周，稳定每月 |

### 5.2 数据质量要求

```yaml
必填字段:
  - 指标数值
  - 采集日期时间
  - 采集方式（手动/OCR）

血药浓度额外字段:
  - 药物类型
  - 上次服药时间
  - 采血时间

异常值提示:
  - 超出正常范围标记
  - 与历史数据对比提示
  - 单位自动校验
```

| 术语 | 英文 | 解释 |
